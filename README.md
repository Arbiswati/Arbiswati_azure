Azure Hosting Project
Project Overview

This repository contains documentation and files related to my Azure project on hosting a website and setting up a Virtual Private Cloud (VPC) connection. The purpose of this project is to demonstrate the deployment of a web application on the cloud and proper network configuration.

Files in the Repository:-----

Project on hosting website.pdf: Step-by-step guide on hosting a website on Azure.

VPC Connection.pdf: Details and configuration of VPC connection for secure networking.

Technologies Used:----

1.Microsoft Azure

2.Virtual Private Cloud (VPC)

3.HTML / CSS / JavaScript (for the website)

Key Features:------

1.Successfully hosted a website on Azure

2.Configured a secure VPC connection

3.Demonstrated cloud networking concepts
